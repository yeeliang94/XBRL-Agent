PwC Design System

A lightweight, minimal component and style reference for standardizing application interfaces. Built on PwC's 2025 brand foundations — restrained typography, generous whitespace, and a disciplined black-and-orange palette.

Overview

This document defines the visual language and reusable interface components for internal and client-facing applications. It is organized into Foundations (the design tokens everything is built from) and Components (the assembled UI patterns). Every example is live HTML/CSS that maps directly to the CSS custom properties defined at the top of this file.

On brand accuracy: The colour values in this system are taken directly from PwC's official brand standard at brand.pwc.com/standards/colour.html. The signature orange is #FD5108 (Pantone 1655C), supported by the official orange tint ladder (400–50), with white and black as core colours. Status colours (#059669, #E9B01F, #DC2626) are for functional UI states only and are not brand colours. The neutral grey scale is a UI-structural extension, not part of the core brand palette. Typography was not included in the supplied pages — replace --font-sans with PwC's official typeface (e.g. the corporate "Charter"/"Helvetica Neue" pairing per your brand portal) when available.

Design Principles

Four ideas guide every decision in this system.

Light & open

White surfaces, soft borders, and generous spacing. Weight is created by space, not by heavy fills.

Restrained type

Light and regular weights carry most of the interface. Emphasis is earned through hierarchy and color, not bold everywhere.

Orange with intent

The signature orange marks the single most important action or status in a view — never decoration.

Quietly consistent

One radius scale, one spacing rhythm, one shadow family. Predictability is the feature.

Color

Per PwC's brand standard, orange is a single signature accent used to "leave our mark" — we lead with orange, never as full-bleed fills, and reach into the orange tints when more range is needed. White carries backgrounds; greys handle structure. Status colours are functional only and are not brand colours.

Typography

A single sans-serif family across the system. Hierarchy comes from size and weight contrast — large headings sit at light (300) weight, body stays regular (400), and bold is reserved for genuine emphasis only.

Token Usage

All components reference CSS custom properties. Theming the entire system — or swapping in official PwC values — is a matter of editing the :root block.

/* Replace these two with official corporate tokens */
:root {
  --pwc-orange: #FD5108;   /* core orange · Pantone 1655C */
  --font-sans:  "YourCorporateFont", "Helvetica Neue", Arial, sans-serif;
}

/* Then everything inherits automatically */
.btn-primary { background: var(--pwc-orange); }
.input:focus { box-shadow: 0 0 0 3px var(--pwc-orange-soft); }
